* DLU対応
  - o：すでにDLU対応済み
  - ->o：現在 DLU未対応だが、DLU対応不要と判断
      - 例：shapeなどのTensorの形状値を返す者等
* API Section
  - APIの区分
      - TF公式リファレンスページの"See the guide:"の部分の分類
* fjlibステータス
  - `fjxxx`：既存(0.8版)のAPIで存在する
  - `-fjxxx`：既存(0.8版)のAPIで存在しないが、ライブラリ側の修正で簡単に追加できそうなもの
  - `-のみ`：明らかにfjlibの対応は不要と思われるもの
      - 例：shapeなどのTensorの形状値を返すもの、_HostRecvとかCopyのみのもの等
  - `?`：fjlib対応が必要かも？
* Helpステータス
  - `o`：要調査(一次調査でよくわからなかった)
  - `ok`：レビュー＆議論済み
  - `o?`：要精査(レビュー後 要精査と判断)
